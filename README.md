# 🥚 洛克王国精灵蛋互换站

这是我在 **Santa Fe College** 学习期间开发的第一个开源项目，旨在帮助《洛克王国》玩家高效互换精灵蛋。

## ✨ 项目功能
* **信息发布**：快速发布你想交换的宠物和 UID。
* **数据持久化**：使用 JSON 文件存储所有交换请求。
* **响应式设计**：支持手机和电脑访问。

## 🛠️ 技术栈
* **后端**: Python (Flask)
* **前端**: HTML/CSS
* **部署**: Render + GitHub
